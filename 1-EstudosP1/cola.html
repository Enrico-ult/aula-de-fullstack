🏗️ Estrutura Básica HTML

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8"/>
  <title>Título</title>
  <link rel="stylesheet" href="style.css"/>
</head>
<body>
  ...
</body>
</html>

📐 Flexbox

.container {
  display: flex;
  flex-direction: row;    /* ou column */
  justify-content: space-between;
  align-items: center;
  gap: 16px;
  flex-wrap: wrap;
}

.item {
  flex: 1;       /* ocupa espaço igual */
  flex-shrink: 0; /* não encolhe */
}

🔲 CSS Grid

.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  /* ou: repeat(3, 1fr) */
  /* ou: repeat(auto-fill, minmax(200px,1fr)) */
  gap: 20px;
  grid-template-rows: auto;
}

/* Fazer item ocupar 2 colunas: */
.item-grande {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}

🎯 Posicionamento

/* Pai precisa ter position: relative */
.pai { position: relative; }

/* Filho flutua sobre o pai */
.filho {
  position: absolute;
  top: 10px;
  right: 10px;
}

/* Fica fixo na tela ao rolar */
.fixo { position: fixed; top: 0; }

/* Empurra outros elementos normalmente */
.normal { position: relative; top: 5px; }

🎨 Seletores Importantes

/* Tag */
p { color: red; }

/* Classe */
.card { background: white; }

/* ID */
#logo { font-size: 24px; }

/* Pseudo-classes */
a:hover { color: blue; }
li:first-child { font-weight: bold; }
li:last-child { border: none; }
li:nth-child(2) { color: green; }

/* Pseudo-elementos */
p::before { content: '→ '; }
p::after  { content: ' ←'; }

📦 Box Model

.caixa {
  width: 200px;
  height: 100px;

  padding: 10px;       /* espaço interno */
  border: 2px solid;   /* borda */
  margin: 20px;        /* espaço externo */

  /* Padding não aumenta o tamanho total: */
  box-sizing: border-box;
}

🌈 Cores e Backgrounds

/* Formas de declarar cor */
color: red;
color: #ff0000;
color: rgb(255, 0, 0);
color: rgba(255, 0, 0, 0.5); /* com opacidade */

/* Background */
background-color: #f0f0f0;
background-image: url('img.jpg');
background-size: cover;
background-position: center;

/* Gradiente */
background: linear-gradient(135deg, #667eea, #764ba2);

✍️ Tipografia

p {
  font-size: 16px;
  font-weight: bold;   /* ou 400, 700 */
  font-style: italic;
  line-height: 1.6;
  text-align: center;  /* left, right, justify */
  text-transform: uppercase;
  letter-spacing: 2px;
  color: #333;
}

/* Fonte externa (no <head>): */
/* <link href="fonts.googleapis.com/css2?family=Roboto"> */
body { font-family: 'Roboto', sans-serif; }

⏱️ Estratégia para 80 minutos

  1. 0–5 min → Leia o enunciado, planeje a estrutura
  2. 5–15 min → Escreva o HTML completo (cabeçalho, main, footer)
  3. 15–55 min → Escreva o CSS (reset → layout → componentes)
  4. 55–70 min → Ajustes visuais, hover states
  5. 70–80 min → Revisão final, valide links e classes

✅ Sempre começa pelo * { box-sizing: border-box; }

✅ Use max-width + margin: 0 auto para centralizar

✅ Flexbox para Nav e Sidebar, Grid para galeria/cards

✅ Nomeie classes em português e com sentido claro
